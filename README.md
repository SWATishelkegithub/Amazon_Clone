🛒 Mini Amazon Clone

📌 Project Overview
The Mini Amazon Clone is a full-stack e-commerce web application that allows users to browse products, add items to cart, and place orders. It also provides basic admin functionalities to manage products and users.
This project is developed to demonstrate full-stack development using modern technologies and follows structured software development practices.


🛠️ Tech Stack
🔹 Frontend
React.js
Axios
React Router
CSS
🔹 Backend
Spring Boot
REST APIs
MySQL
JPA / Hibernate

✨ Features
👤 User Features
Browse all available products
View product details
Add products to cart
Remove products from cart
Place orders

🔐 Admin Features
Add new products
View all products
Delete products
Manage product details

🚀 How to Run the Project
🔹 Backend Setup
Open backend folder
Configure MySQL database in application.properties
Run Spring Boot application

Backend runs on:
http://localhost:8080
